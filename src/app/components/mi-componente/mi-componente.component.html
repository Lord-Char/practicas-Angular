Practica de creacion de componentes en Angular

En este ejercicio, se practica la creacion de nuevos componentes

con nuevos ficheros, para la aplicacion web

El problema de este metodo es que puede ser repetitivo,

por lo que se usan comandos en la consola de la app

El nuevo component generado por consola

es "peliculas"

Este es un parrafo de prueba con la nueva estructura

La ventaja de separar del component el template en html es que

puedo trabajar sin mezclar la logica de JS con el HTML

Es decir, separo el componente de logica JS del componente de la vista en html
